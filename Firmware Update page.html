Update Instructions

Step 1: Download Firmware

Download the latest firmware from the link above.

Step 2: Connect Device

Connect your Universal Programming Kit to your computer using a USB cable.

Step 3: Run Update

Open the programming software, go to the 'Firmware Update' section, and upload the downloaded firmware file.

Step 4: Complete

Follow the on-screen instructions to complete the update process. Once done, restart the device.

Firmware Compatibility

The latest firmware version is compatible with the following radio models:

  • Motorola GP300
  • Kenwood TK-480
  • Baofeng UV-5R
  • Hytera PD785
  • Icom IC-F4001
