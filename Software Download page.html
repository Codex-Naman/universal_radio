Installation Instructions

Windows Installation

  1. Download the installer from the link above.
  2. Run the setup file and follow the on-screen instructions.
  3. Restart your computer after installation is complete.

macOS Installation

  1. Download the macOS installer.
  2. Double-click the .dmg file to mount the disk image.
  3. Drag the application to the Applications folder.
  4. Open the application and follow the setup guide.

Linux Installation

  1. Download the .tar.gz file for your distribution.
  2. Extract the files in the terminal using the command tar -xvzf.
  3. Run the install script with sudo ./install.sh.
  4. Follow the on-screen instructions to complete the setup.
